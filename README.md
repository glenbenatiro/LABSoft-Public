# LABSoft-Public
LABSoft is the complementary software application to the Lab in a Box, a portable and low-cost electronics trainer kit powered by a Raspberry Pi Zero.

This is a public mirror of the LABSoft private repository.

LABSoft is developed using C++, with Fast Light Toolkit (FLTK) for its graphical user interface.

### Raspberry Pi Compatibility
- Raspberry Pi Zero
- Raspberry Pi Zero W
- Raspberry Pi Zero 2 W
- Raspberry Pi 3 Model B+
- Raspberry Pi 4 Model B
